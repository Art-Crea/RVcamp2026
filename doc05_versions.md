## GitHub でのタグ打ち方

```bash
git tag v1.2
git push origin v1.2
```

GitHub の「Releases」→「タグを作成」でリリースノートも書けます。

---

## Notion への取り込み

このファイル（Markdown）をNotionに取り込む方法：

1. Notion のページを開く
2. 「+」→「インポート」→「Markdown & CSV」
3. このフォルダの `docs/*.md` ファイルを選んでアップロード

または GitHub 連携を使う場合：
- Notion Integration を使って GitHub リポジトリと同期
