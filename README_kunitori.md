## 地図の切替（運営のみ）

運営URLで開き、「🔐 運営」タブへ。

- **地図の切替ボタン**（OSM / Google）→ その端末だけ即切替
- **APIキー入力** → その端末の localStorage に保存。1度入れれば以後は不要
- **「誰の画面に何が見えるか」の表**が常に出ます

## 本番固定

決めたら「🔒 本番用に固定する」で **全員OSM** か **全員Google** を押すと、`rv_config.js` に貼る3行が出力されます。コピーしてコミットすれば固定完了。以後は**運営の端末上書きも無視され、全員がその設定**になります。

### 挙動の全パターン

| `rv_config.js` の設定 | 運営端末 | 参加者スマホ |
|---|---|---|
| 初期状態（MAP_MODE:'osm', GMAP_KEY:'', LOCK_MAP:false） | OSM | OSM |
| 運営がGoogleキーを端末に保存して試用 | **Google** | OSM |
| 本番固定：`MAP_MODE:'osm', LOCK_MAP:true` | OSM | OSM |
| 本番固定：`MAP_MODE:'gmap', GMAP_KEY:'AIza…', LOCK_MAP:true` | **Google** | **Google** |

**⚠ 構造上の事実：参加者全員にGoogle地図を見せるには、キーを `rv_config.js`（＝公開ファイル）に書くしかありません。** キーがGM端末のlocalStorageにしか無い場合、Google地図が出るのはGM端末だけです。

## ⚠ Googleキーを `rv_config.js` に入れる前に（必須・5分）

クライアントサイドの地図キーは、開発者ツールのNetworkタブに**必ず現れます**。UIで隠しても秘密になりません。
守り方は「隠す」ではなく「縛る」です。[Google Cloud Console](https://console.cloud.google.com/google/maps-apis/credentials) で：

1. **アプリケーションの制限** → ウェブサイト → `https://art-crea.github.io/*`
2. **APIの制限** → キーを制限 → **Maps JavaScript API** のみ
3. **お支払い → 予算とアラート** → 上限を設定

これで、キーが第三者に見られても `RefererNotAllowedMapError` で弾かれ、他人には一切使えません。**この設定が先、コミットが後。**

## 事故防止

キー未設定・認証拒否・通信断・8秒無応答 → **すべて自動でOSMにフォールバック**。当日、地図が真っ白になって試合が止まることはありません。理由もチップに表示されます。

## モードA：GMが1台で操作

準備ゼロ・ネット不要（地図タイル以外）。GM本陣で 方式＝**ローカル**、手番方式＝**交互手番**。
盤のパネル／地図のピンをクリックで占領。挟んだ敵陣は自動反転、手番も自動交代。**当日の最終フォールバック**にもなります。

## モードB：36人が各自スマホ

手番方式＝**早取り制**（36人で交互手番は成立しないため）。

### 同期バックエンド（現地で切替可）
- **Googleスプレッドシート＋Apps Script** — 遅延3秒／無料／運営がシートを見れば戦況が丸見え／ログが自動で溜まる。手順は `rv_gas_backend.gs` の冒頭コメント
- **Firebase Realtime Database** — 遅延1秒未満／無料枠で十分

### 判定（開戦中でも切替可）
- **① GPS圏内（既定80m）で即占領** — 速い／不正の余地あり
- **② GPS＋ヒント回答 → GM承認** — 推奨
- **③ 現地写真＋GM承認** — 最も厳格／GM負荷大

②③はGM本陣「承認キュー」に、距離・回答・写真・申請地点の地図リンクが1画面で並びます。渋滞したら「圏内の申請をまとめて承認」で一括処理、それでも詰まったら①に落とせます。

### 参加者導線
GM本陣「参戦リンクを作る」→ QR化して武将カードに印刷 → 読むと自動で同じルームに接続 → 武将を選び暗証番号 → 出陣

## 当日リスクと対策

| リスク | 対策 |
|---|---|
| Google地図キーが弾かれる | **自動でOSMに落ちる**。試合は止まらない |
| 山間部（鶴見岳・城島高原）で電波が切れる | 圏内に戻ってから申請が送られる。不安ならこの2陣所はGM代理占領に |
| GPS誤差が大きい | 判定半径 80m→150m にその場で変更（GM本陣） |
| 承認が渋滞 | 判定を②→①に落とす |
| 全部落ちる | ローカルモードでGM1台に切り替え、盤だけ回す |

**GM端末は必ず2台**（1台は予備、同じ運営URLを開いておく）。

## 25陣所の状態

**確定 20件**（Google Places 実座標）／**候補 5件**（別府市役所・堀田温泉・志高湖・城島高原パーク）。
差し替え用の**控え6件**（鉄輪むし湯／地獄蒸し工房／湯けむり展望台／柴石温泉／別府海浜砂湯／うみたまご）を運営タブからワンクリックで投入できます。

⚠ **座標はGoogle Places由来の代表点です。当日の集合地点（入口・駐車場）は必ず現地で確認し、地図上のピンをドラッグして修正してください。** 全端末に即反映されます。

GOALのグランシア別府鉄輪から最遠は**浜脇温泉で 6.3km**。25全制圧は物理的に不可能なので、勝敗は「時間内にどれだけ盤を染めたか」で決まります。挟み反転があるので、終盤に1枚取って一気にひっくり返す逆転劇が起きる設計です。
