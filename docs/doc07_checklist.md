## 緊急時対応

| 症状 | 対応 |
|---|---|
| Google地図が出ない | 自動でOSMに切替済み。試合継続 |
| GAS同期が止まった | GM本陣→「ローカル」切替。GM1台で占領操作 |
| 電波がない陣所 | 圏内に戻ってから申請。GMが代理占領でも可 |
| 承認が渋滞 | 判定を「GPS即占領」に変更（GM本陣→ルール） |
| 全部落ちた | ローカルモード+GM1台で盤だけ回す |
