🚀 次にやること（優先順）

① ADMIN_TOKEN を自分だけの文字列に変更
rv_config.js の ADMIN_TOKEN: の値を変更してコミット。現在は初期値から変更済みですが、さらに自分だけが知る文字列にしてください。
② このZIPを GitHub にアップロード（iPhone Safari 可）
github.com/Art-Crea/RVcamp2026 → Add file → Upload files → ZIPを展開してアップロード。手順は doc08_github.md と doc10_iphone.md を参照。
③ GitHub Pages を有効化
リポジトリ Settings → Pages → Source: main / ルート → Save。5分で https://art-crea.github.io/RVcamp2026/ が公開されます。
④ GAS に rv_gas_backend.gs をデプロイ
Googleスプレッドシート → 拡張機能 → Apps Script → コードを貼付 → デプロイ（全員アクセス可）。URLをGM本陣の接続欄に貼る。
⑤ LINE 公式アカウントを接続（doc04_line.md 参照）
LINE Developers でチャネル作成 → トークン発行 → GASスクリプトプロパティに設定 → Webhook URL に ?key=合言葉付きURLを貼る。
⑥ スマホ実機テスト（doc07_checklist.md チェックリスト）
rv_player_app.html をスマホで開き GPS許可 → 陣所に近づいて申請ボタンが出るか確認。8/27の2週間前までに完了推奨。
⑦ Googleマイマップ作成（参加者ナビ用）
合戦盤「陣所一覧・書出」→ CSV書出 → Googleマイマップにインポート → 共有URL を rv_config.js の MYMAP_URL に貼る。
⑧ 本番2週間前：36将QRカードを印刷
GM本陣「参戦リンクを作る」→ 36将分のQRを生成 → A4カード印刷。当日は受付で配布。

📱 iPhone からの操作

Safari で GitHub.com を開く（推奨・無料）
github.com/Art-Crea/RVcamp2026 → Add file → Upload files → ファイルアプリからZIP展開後のフォルダを選択。100ファイルまで1回でOK。
Working Copy アプリ（本格git・¥2,200）
App Store で「Working Copy - Git Client」をインストール。Personal Access Tokenを発行してclone → 編集 → commit → push が全部iPhoneで完結。手順は doc10_iphone.md を参照。
1ファイルだけ編集（例：ADMIN_TOKEN変更）
GitHub.com → rv_config.js → 鉛筆アイコン → 編集 → Commit changes。Safariだけで完結。
